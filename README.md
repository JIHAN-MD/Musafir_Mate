# Musafir Mate

## Project Overview

Musafir Mate is a Laravel-based web application developed to assist Muslim travelers in planning halal-friendly trips. The platform provides travel information, halal-certified accommodations, restaurants, travel packages, flights, prayer facilities, and booking services.

## Features

* User Registration and Login
* Destination Management
* Hotel Listings
* Halal Restaurant Listings
* Travel Package Management
* Flight Search and Booking
* Prayer Time Information
* Qibla Direction Assistance
* User Dashboard

## Technology Stack

* Laravel Framework
* PHP
* SQLite / MySQL
* HTML
* CSS
* Bootstrap
* JavaScript
* XAMPP

## Installation

Clone the repository:

```bash
git clone https://github.com/JIHAN-MD/Musafir_Mate.git
```

Install dependencies:

```bash
composer install
npm install
```

Configure environment:

```bash
copy .env.example .env
php artisan key:generate
```

Run migrations and seeders:

```bash
php artisan migrate:fresh --seed
```

Start the server:

```bash
php artisan serve
```
